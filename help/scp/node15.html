b.
Es posible consultar a expertos: Si los expertos están disponibles, se puede diseñar con elllos una adecuada; mediante encuestas o técnicas más sofistricadas como las entrevistas Delphi, se puede intentar dar respuesta a las siguientes preguntas:

i.
¿La relación entre entradas y salidas es monótona?
ii.
¿En caso afirmativo, cuál es el sentido de esa monotonía, creciente o decreciente?
iii.
¿Qué tan importante es cada entrada?
iv.
¿La salida debe estar4 centrada cuando las entradas son las mínimas posibles?
v.
¿Deberiamos esperar una relación lineal para cada entrada?
